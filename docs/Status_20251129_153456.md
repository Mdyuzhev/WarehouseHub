## Выполненные задачи

### 1. Redis (WH-66, WH-67) ✅

**K8s манифесты созданы:**
- `k8s/warehouse/redis-deployment.yaml` - Redis 7-alpine, 256Mi memory
- `k8s/warehouse/redis-service.yaml` - ClusterIP на порт 6379
- `k8s/warehouse/redis-configmap.yaml` - maxmemory 200mb, allkeys-lru

**Интеграция в warehouse-api:**
- `RedisConfig.java` - конфигурация кэша с TTL 5 минут для products
- `RateLimitingService.java` - rate limiting: 5 попыток за 15 минут
- `TokenBlacklistService.java` - инвалидация JWT при logout
- `ProductService.java` - добавлены `@Cacheable`, `@CacheEvict`
- `AuthController.java` - проверка rate limit, endpoint `/api/auth/logout`
- `JwtAuthenticationFilter.java` - проверка blacklist перед аутентификацией

**Важные решения:**
- Сервисы Redis опциональны (`@ConditionalOnBean`, `@Autowired(required=false)`)
- `spring.cache.type=redis` включается только в K8s профиле
- Это позволяет тестам работать без Redis
