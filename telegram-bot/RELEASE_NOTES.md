# Release Notes — Warehouse Telegram Bot

## v5.2.0 (2025-12-01)

### Новые возможности
- **🤖 Warehouse Robot** — управление эмуляцией складских операций
  - Сценарии: Приёмка, Отгрузка, Инвентаризация
  - Продолжительность: 5 мин / 30 мин / 1 час / однократно
  - Скорость: пауза между повторами (1с / 5с / 15с)
  - Планирование запуска по расписанию (время МСК)
  - Уведомления с детализацией товаров (название, количество, цена)
  - Остановка запущенного сценария
  - Просмотр статуса и статистики робота
- **Улучшенные уведомления** — детальная информация о товарах в каждом сценарии
- **Поддержка STAGING/PROD** — выбор окружения для запуска сценариев

### Изменения
- Версия бота обновлена до 5.2.0
- Обновлена справка /help с описанием Robot функций
- Обновлён /release с новой информацией

### Технические детали
- Новый сервис: `services/robot.py` — клиент для Warehouse Robot API
- Новый handler: `handlers/robot.py` — обработчики команд робота
- Новый сервис: `warehouse-robot/` — FastAPI сервис эмуляции склада
- Обновлены: keyboards.py, commands.py, app.py, messages.py

## v5.0.0 (2025-11-25)

### Новые возможности
- Полная интеграция с GitLab CI/CD
- Управление деплоем через бота (staging/production)

- Мониторинг K8s подов и метрик

- Telegram Long Polling
- GitLab Webhooks
- Locust integration
- Allure reports
